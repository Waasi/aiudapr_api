# AiudaprApi

This is the Graph QL API for AiudaPR.
AiudaPR is a web application for the
notification of help and utility spots.
Find more about AiudaPR [here](https://github.com/makobi/aiudapr)

# Setup

- `mix deps.get`
- `make init-psql`
- `make start-psql`
- `make setup-psql`
- `make migrate`
- `make server`
